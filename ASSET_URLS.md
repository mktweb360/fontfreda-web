## Notas Técnicas

- **Resolución:** Todas las imágenes están en 2560x1440px (16:9), optimizadas para hero sections
- **Formato:** PNG original + WebP comprimido para mejor rendimiento web
- **Uso recomendado:** Utilizar URLs comprimidas (WebP) en producción para mejor velocidad de carga
- **Ciclo de vida:** Las URLs están vinculadas al proyecto Fontfreda.net y persistirán mientras el sitio exista
- **Contraste de texto:** Las imágenes tienen tonos cálidos (dorados, verdes, naturales). Usar texto oscuro o blanco con suficiente contraste
